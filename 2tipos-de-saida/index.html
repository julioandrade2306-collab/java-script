Tipos de saída em javascript

Vamos aprender as diferentes formas de exivir formações com javascript

  1. 1. Exibir através de um elemento HTML usando getElementById

    Utilizamos document.getElementById para acessar o elemento com um ID específico

    Em seguida, usamos TextContent ou InnerHTML para mostrar o conteúdo

  2. 2. Exibir com document.write()

    Este método escreve diretamente no corpo da página. Não é recomendado para uso moderno, mas ainda funciona.

  3. 3. Exibir com alert()

    Este comando exibe uma janela de alerta para o usuário

  4. 4. Exibir com console.log()

    Este método exibe mensagens no console do navegador (F12 aba console).
